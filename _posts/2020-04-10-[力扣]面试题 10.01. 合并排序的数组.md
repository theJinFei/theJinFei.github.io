## 题目描述
>   给定两个排序后的数组 A 和 B，其中 A 的末端有足够的缓冲空间容纳 B。 编写一个方法，将 B 合并入 A 并排序。
    初始化 A 和 B 的元素数量分别为 m 和 n。



## Example1:
 
>       输入:
        A = [1,2,3,0,0,0], m = 3
        B = [2,5,6],       n = 3
        输出: [1,2,2,3,5,6]

## 提示
- A.length == n + m


## 解题思路
- 从后往前挪动指针就可
- 最后有可能B剩下，单独处理即可



```C++
class Solution {
public:
    void merge(vector<int>& A, int m, vector<int>& B, int n) {
        int size = m + n;
        int i = m - 1;
        int j = n - 1;
        int end = size - 1;
        while(j >= 0 && i >= 0){
            if(A[i] > B[j]){
                A[end--] = A[i];
                i--;
            }else{
                A[end--] = B[j];
                j--;
            }
        }
        while(j >= 0){
            A[end--] = B[j--];
        }
    }
};
```
